# Language Phonetic Analysis Tools

## Functionalities
Extract syllables from phonetic sequence(sapi) of an American English word

## Example
Run:
```python
py ./py/example.py
```

Returns:
`Word 'communication' has following syllables:  dict_keys(['K_AH_M', 'M_Y', 'Y_UW_N', 'N_AH', 'K_EY', 'SH_AH_N'])`
